Auto Lead Finder

Searches Google directly and automatically lists the tradies in an area who have no website — ready to call.

❓ How to get a free Google API key (one-time, ~10 min)
  1. Go to console.cloud.google.com and sign in with your Google account.
  2. Top bar → New Project → name it (e.g. "ReviewTap") → Create.
  3. Enable billing: menu → Billing → link a billing account → add a card. Google gives $200 free every month — normal use won't cost you a cent, but a card is required.
  4. Menu → APIs & Services → Library → search and Enable both: “Places API (New)” and “Maps JavaScript API”.
  5. Menu → APIs & Services → Credentials → Create Credentials → API key. Copy it.
  6. (Recommended) Click the key → Restrict it: Application restrictions → Websites → add your Netlify link and localhost. API restrictions → tick the two APIs above.
  7. Paste the key above and hit Save key.

No key yet? Use the free manual Lead Finder in the meantime. · Data from Google Places.
